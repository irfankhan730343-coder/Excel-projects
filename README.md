# Customer Support Dashboard Project in Excel

## Project Overview

The Customer Support Dashboard is an interactive Excel dashboard developed to analyze customer support operations, ticket handling performance, customer satisfaction ratings, and agent productivity. The dashboard transforms raw customer support data into meaningful business insights using advanced Excel tools and visualization techniques.

The primary goal of this project is to monitor customer interactions, track ticket resolution performance, evaluate agent efficiency, and improve overall customer service quality through data-driven analysis.

This project demonstrates practical knowledge of Excel dashboard development, business reporting, and data visualization techniques commonly used in customer support and business analytics domains.

---

# Dataset Information

The dataset used in this project contains customer support-related information with the following columns:

* Customer ID
* Order ID
* Customer Name
* Contact Date
* Contact Type
* Is It for Return
* Ticket ID
* Agent Handled
* Rating Given

The dataset was cleaned, organized, and transformed before creating the dashboard to ensure accurate reporting and analysis.
